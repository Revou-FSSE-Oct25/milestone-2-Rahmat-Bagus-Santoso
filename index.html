About Carbonsdale

Carbonsdale is a fictional game studio built as a personal learning project. The focus is not on flashy visuals, but on writing clear, maintainable code and understanding how small decisions affect the overall structure.

Each game is developed step by step, with logic and UI separated intentionally. Features are only added after the core structure is stable and decisions are locked.

This project is a space to practice discipline, refactoring awareness, and building things that can be safely left and revisited later without fear.
