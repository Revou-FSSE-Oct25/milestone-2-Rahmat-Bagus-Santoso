# Carbonsdale

Carbonsdale is a fictional game website that features a collection of simple and interactive web-based games built using JavaScript.

This project is created to practice basic web development concepts such as HTML structure, styling with Tailwind CSS, and JavaScript logic.

### Live Demo
**Github Pages** https://revou-fsse-oct25.github.io/milestone-2-Rahmat-Bagus-Santoso/

## Available Games

- **Guess the Number**  
  Try to guess a randomly generated number within a limited number of attempts.

- **Rock Paper Scissors**  
  Play a classic Rock–Paper–Scissors game against the computer.

- **Clicker Game**  
  Click as many times as possible before the timer runs out.

## Tech Stack

- HTML5 (Semantic HTML)
- Tailwind CSS
- JavaScript (Vanilla / DOM)
- Node.js (for Tailwind CLI)
- Ai (Chatgpt)
